# 대덕산업 홈페이지 배포용 파일

## 폴더 구조
- index.html : 문구/섹션 수정
- assets/css/style.css : 색상/간격/폰트 등 디자인 수정
- assets/images/ : 홈페이지에 쓰이는 이미지 파일들

## GitHub 업로드 방법
1. 이 폴더 안 파일을 모두 GitHub 저장소에 업로드합니다.
2. 기존 index.html 이 있으면 덮어쓰기 합니다.
3. assets 폴더도 함께 업로드합니다.
4. Vercel이 연결되어 있으면 자동 재배포됩니다.

## 수정 방법
### 문구 수정
- index.html 열기
- 원하는 문장 수정 후 저장

### 사진 교체
- assets/images 안 사진 파일을 같은 이름으로 바꾸거나
- index.html 안 파일명을 새 이름으로 수정

### 디자인 수정
- assets/css/style.css 열기
- 색상, 여백, 글자크기 수정
